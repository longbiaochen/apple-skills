# Apple Notes Reference

## Common Commands

```bash
memo notes
memo notes -f "Folder Name"
memo notes -s "query"
memo notes -a "Note Title"
memo notes -e
memo notes -m
memo notes -d
memo notes -ex
```

## Workflow Notes

- Search before edit or delete when the note title is ambiguous.
- Some `memo` flows are interactive and may not fit a fully non-interactive workflow.
- Notes with images or attachments may not support full edit flows.
